# carsten-coffee-api

Golang implementation of COFFEE API.

## Installation

Assuming you have go, do

```
go get -u gitlab.nsd.no/nsd-utvikling/carsten-coffee-api
```

Then install dependencies with [Govendor](https://github.com/kardianos/govendor).

```
govendor sync
```

## Execution

Simply run

```
go run .
```

and access
```
http://localhost:1337
```
